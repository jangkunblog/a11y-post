## 수집 시 주의사항

1. **기간 필터링**: 반드시 지난 1주일 이내 콘텐츠만 포함
2. **중복 제거**: 기존 `a11y-weekly-*` 포스트에 이미 포함된 링크/주제 제외
3. **신뢰성**: 공식 소스(W3C, 정부 기관, 인증 기관) 우선
4. **균형**: 글로벌 뉴스와 국내 뉴스를 균형 있게 포함
5. **실용성**: 개발자가 실무에 적용할 수 있는 내용 우선
6. **최소 항목**: 카테고리별 최소 1개 항목 목표 (없으면 "주요 업데이트 없음" 표기)
7. **최대 항목**: 카테고리별 최대 5개 (중요도순 선별)
8. **카카오 접근성 맥락**: 카카오 관련 검색 시 아래 키워드를 참고하여 관련성 판단
   - 카카오톡 고대비 테마, 이모티콘 대체 텍스트
   - 카카오맵 교통약자 정보 (승강장 연단, 휠체어 경로)
   - 카카오게임즈 플레이버디 (보조기기 지원)
   - 링키지랩 (장애인 표준사업장)
   - if(kakao) 컨퍼런스 접근성 세션
